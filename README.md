# ScannerJammer
*Stop scanning my Minecraft server!*

## Summary
Bans any player from your Minecraft server with "scan" in their name.  
I realize this is completely useless because as soon as the account joins,  
it's already gotten all of the information it wants,  
but seeing three different scanners join my server made me angry.

## Usage
Once this plugin is installed, it does all of the work.  
You can exempt any username by issuing command `/scannerjammer:exempt username`.  
(This feature is untested because my server doesn't get traffic lol)

## Installation
Check the Releases tab on the right and download the JAR file.  
Place the plugin file in your `plugins` folder.  
Restart your server.

## Building
Add the project files to your favorite IDE.  
Build with Maven.  
(I have never used Maven and I rarely code in Java lol)
